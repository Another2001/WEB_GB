На сайте https://ru.wikipedia.org/ содержатся следующие блоки:

1) "Шапка". Содержит в себе элементы для авторизации пользователя, строка поиска(отдельные страницы по ключевым словам внутри сайта),
 просмотра кода страницы, перехода во вкладки обсуждений / истории изменений(как commit в GIT`e)

2) Блок основного контента, в котором множество более мелких блоков для структурирования и раздельной стилизации контента. Контент разделен
 в две колонки, каждый "кусочек" информации расположен отдельно в стилистической рамке и снабжен элементами JavaScript 
 (кнопки, всплывающие меню / подсказки). Также отделен крупный блок для навигации по сайту для быстрого и удобного использования функционала сайта
 (ссылки на оф. форум и сообщество / последние созданные статьи / справка / переход на другой язык).

3)"Подвал" содержит информацию по юридическим и правовым вопросам, ссылку на мобильную версию сайта, статистику, а также информацию для разработчиков.
